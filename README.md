# Memory Game (Vue.js)

This project is a simple memory card game built with Vue.js.

The goal of the game is to find matching pairs of cards by flipping them two at a time.

## Features

- Flip cards
- Match pairs of cards
- Simple game logic
- Interactive interface

## Technologies

- Vue.js
- JavaScript
- HTML
- CSS
- Vite

## Purpose

This project was created to practice front-end development and learn how to build interactive applications with Vue.js.

⚠️ This project is still a work in progress.

## Installation

1. Clone the repository

2. Install dependencies

3. Run the project
